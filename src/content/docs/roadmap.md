# The Roadmap
Welcome to the roadmap page. While Jarvix is functional and can achieve its core mission, we have not the achieved the full vision of what it can do. Thus, below we are sharing our roadmap.

> Note that the roadmap below is **not** in order, is **not** based on priority and it **may** change over time as we receive feedback from you -- our users. The road map

## UI/UX changes
* Add a method for user to set a default root folder for all their projects.
* Add multiple themes (including light & dark) to allow personalization.
* Add shortcuts for starting, pausing, stoping the pipeline.
* Add tooltips to different buttons.
* Link to website docs.
* Manipulate agents in pipeline.
* Allow defining and editing agents in app.
* An agent editor inside Jarvix
* Allow user to see what the AI is doing

## Project
* Move agents who are added to the pipeline from global agents folder to local.
* Allow agent templates.

## Core
* Phase 1: Allow other CLIs; candidates include Claude Code and Kilo.
* Phase 2: Allow CLI per agent.
* Allow sub-agent clusters.
* Research differentiated prompts (agent prompt vs regular)
* Research dockerized producer and fixer agents + yolo.
* Surface AI WIP

## Website
* Enhance the getting started guide.
* Update it based on the above changes.
* Agent marketplace
